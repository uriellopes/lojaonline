#### DESCRIÇÃO
Projeto de uma loja online ( modelo Amazon, Mercado Livre, etc...) onde o usuário poderá buscar um produto, que foi anteriormente cadastrado por uma loja, e fazer um pedido desse produto.

#### VERSÕES
- Java JDK 21
- Maven 3.9.6
- MySQL 8
- Spring 3.2
